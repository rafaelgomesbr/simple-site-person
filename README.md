# Requirements
1. [Node.js](https://nodejs.org/) installed on the computer.
# Usage
1. Fork the repo and then clone it or download it.
2. First install all dependencies:
    ```bash
    # with npm
    npm install
    
    # or with yarn
    yarn
    ```
    
3. Start the server and access it through the browser:
     ```bash
     npm start
      ```
    * in the browser ->
     localhost:5000
